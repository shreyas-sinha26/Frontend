Design a premium mobile UI for a disaster communication system called "PulseLink".

The UI must feel extremely polished, cinematic, and Apple-level in quality. The design should combine minimalism with futuristic elements while still feeling calm, trustworthy, and reassuring during stressful disaster situations.

DESIGN LANGUAGE
Use a modern product design style inspired by Apple, Linear, Stripe, and Nothing OS interfaces.

The interface should feature:
• Progressive blur glass panels
• Bento grid layout for the dashboard
• Subtle cinematic light rays
• Soft volumetric lighting
• Minimalistic surfaces
• 3D interface elements
• Interactive objects with depth
• Smooth shadows and gradients
• Calm atmosphere that reduces anxiety

VISUAL MOOD
The interface must feel calming, safe, and trustworthy.

Use soft lighting and ambient glow rather than harsh colors.

COLOR PALETTE
Primary background: deep navy gradient (#0B132B → #020617)
Surface panels: translucent glass (#1C2541 with blur)
Accent blue: (#3A86FF)
Calm teal highlight: (#22D3EE)
Emergency red (only for SOS): (#EF4444)
Text primary: soft white (#F8FAFC)
Text secondary: muted gray (#94A3B8)

Use subtle gradient lighting and volumetric light rays emerging from the interface to create depth.

TYPOGRAPHY
Use large, bold typography similar to Apple and modern tech startups.

Font style: SF Pro / Inter style.

Use strong hierarchy:
Large hero headings
Bold labels
Minimal secondary text

Typography should feel confident and readable even in stressful emergency conditions.

LAYOUT SYSTEM
Use a Bento Grid dashboard layout similar to modern Apple product websites.

Cards should have:
• soft rounded corners
• progressive blur
• subtle shadows
• depth layering
• interactive hover states

Each card should feel like a floating glass panel in space.
